How It Went

Start to finish, step by step.

From JAVES's first message to the final videos. Done by hand, just done fast.

1

JAVES reached out

JAVES had just played a rooftop set and wanted videos from the night to post. We got on a quick call to hear exactly what JAVES was going for.

Behind the scenesOn that first call we focused on what JAVES wanted the videos to do, then planned the whole batch around it instead of treating it as a one-off. That clarity up front is what let us move fast later without guessing.
2

JAVES sent songs and reference trends

JAVES came with three video styles in mind, all based on trends going around that JAVES sent over as examples for us to take inspiration from: a short "aura" clip, a longer one cut in time to a song, and a countdown. JAVES picked the songs and sent those too.

How we handled JAVES's feedbackOne thing JAVES wasn't sure about up front was whether to put words on the videos at all. Instead of guessing, we gave JAVES the choice: clean versions to caption later, or the on-screen text handled by us. JAVES also started with fewer videos in mind and grew the idea to three different styles. We said yes to all of it and built the set so the same footage could feed every version.
3

We went through all the footage

We started with all the clips of JAVES from the event. We watched everything and pulled the best moments into clean, organized clips, each one labeled so it is easy to find later.

Behind the scenesThe footage came in mixed: clips of JAVES from different angles and phones, all different lighting and steadiness. We reviewed every second, cut the shaky and dark moments, and kept only frames that actually held up. Each keeper was trimmed tight and labeled by what it shows, so nothing got hunted for twice.
4

We cut the videos to the music

Our team edited each short video by hand so the cuts land right on the beat of JAVES's songs. Nothing generic, every cut is a choice. We just move fast.

Behind the scenesThis is the step we never automate or hand off. An editor decides which moment lands on which beat, by ear and by eye. We build several versions of each idea on purpose, so there are real choices to compare instead of a single take. Fast turnaround, human decisions.
5

We added JAVES's look and words

JAVES's own color style on every clip, plus on-screen text we built from the trends JAVES sent as inspiration: the "aura" look, and a countdown that ticks down on screen.

Behind the scenesJAVES's color style was matched across every clip so the whole set reads as one cohesive body of work, not a pile of random videos. On-screen text was sized and timed to be legible on a phone at a glance, following the format of the trends JAVES referenced.
6

We made longer versions too

Full-length cuts with no text on them, so JAVES could add custom captions anytime.

Why it mattersThe longer cuts run start to finish like a recap with no text baked in. That keeps them flexible: JAVES can post them as-is, or add a caption later to fit a specific platform or moment, without coming back to us.
7

We delivered, days early

A full pack of finished videos, ready to post, handed over in a single shared folder days after the event.

Behind the scenesEverything arrived in one organized, clearly named shared folder alongside the one-page guide. The point is that months from now JAVES can still open it, know exactly what each file is, and post in seconds. Throughout the project JAVES always knew where things stood, so the handoff was a confirmation, not a surprise.

How We Work

Careful with JAVES's work. Quick with ours.

  • Constant status updates. Like a field report to the commander, JAVES always knows the project's health, where we are, and what is coming next. No chasing us for an update.
  • Non-destructive. We only ever edit copies, so JAVES's original videos are never changed or lost.
  • Organized and labeled. Every clip is named by what it shows, so anything can be found in seconds.
  • Version controlled. Every change is saved as its own version, like a full save history, so we can roll back to any point and nothing is ever lost.
  • Beat-matched. Every cut lands right on the beat of the music.
